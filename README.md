# 🛍️ E-Commerce Product Showcase Website

A modern, visually appealing static e-commerce product showcase website built using **HTML, CSS, and JavaScript**. This project focuses on clean UI, smooth animations, and interactive product browsing experience.

---

## 🚀 Features

- 🎨 **Minimal & Premium Design**
  - Clean layout with modern typography
  - White/black theme with accent colors

- 🖼️ **Product Showcase**
  - Responsive product grid
  - Hover zoom effects on product images
  - "New", "Sale", and "Trending" badges

- 🔍 **Quick View Modal**
  - Click on a product to view details in a popup
  - Smooth animations for better UX

- 🛒 **Add to Cart (UI Only)**
  - Interactive add-to-cart button animation
  - Visual feedback for user actions

- ⭐ **Customer Reviews Section**
  - Stylish review cards
  - User testimonials

- 📧 **Newsletter Signup**
  - Email input UI for subscriptions

- 🎞️ **Smooth Animations**
  - Hover effects
  - Transitions and micro-interactions

- 📱 **Fully Responsive**
  - Works perfectly on mobile, tablet, and desktop

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3 (Flexbox / Grid)**
- **JavaScript (Vanilla JS)**
